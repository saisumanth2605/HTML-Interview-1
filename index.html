1.Difference Between Div & Span Tag?

Div:-

  1. Div is a block level Element
  2. Div always Start on the new line & it will takes whole width of the Screen.

Span:-

  1. Span is a Inline level Element
  2. Span takes the Space Required for it,it does not occupy Unecessary Width.

2.Difference Between Class & ID?

Class:-

  1. It called Selector.Class is not Unique,but we can Apply Multiple Classes to any Element.
  2. We denote classes with (.)

ID:-

  1. It called Selector.Id is Unique,but we cannot Apply same value of any id to different element.
  2. In one file there should be just one id with of the same value. & we denote ID's with (#).
